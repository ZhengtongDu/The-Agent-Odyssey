# First Principles Note Template

## Problem

用一句话说明算法、模型或系统要解决的问题。

## Background

说明它出现前的主流方法、瓶颈和关键动机。

## Definition

给出核心对象、符号约定、输入输出和 shape。

## Derivation

写出关键公式，并逐步解释每一步。

示例：

$$
L(\theta) = \frac{1}{2m}\sum_{i=1}^{m}(x_i^T\theta - y_i)^2
$$

## Implementation

给出最小可运行代码，并标注张量维度。

## Strengths And Limits

说明该方法解决了什么、没有解决什么，以及为什么。

## Modern View

结合近期论文、框架或工程实践重新评价这个方法。

## References

- 原始论文或高质量教程。
- 相关实现或可复现实验。

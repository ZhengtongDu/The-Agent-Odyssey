# LeetCode 练习合集

这个合集用于沉淀 6 周学习计划中的算法题训练。题目来源以 LeetCode 官方题单 [Top 100 Liked Questions](https://leetcode.com/problem-list/79h8rn6/) 为主，本地页面只保留题名、链接、主题、排期和个人题解，不复制原题全文。

## Daily Rule

- 每天新任务默认安排 `3-5` 题。
- 新主题日优先安排 `3` 题：1 道 warm-up、1 道主线 Medium、1 道同类变体。
- 复盘日或冲刺日安排 `4-5` 题，按同一模式递进难度。
- 每天的 LeetCode 页面必须包含题目链接、思路、Python 解法、复杂度和边界条件。
- 每天任务页都从本合集进入，并在题单快照里登记计划 Day 和题解入口。

## 每日流程

1. 从 [Top 100 Liked 题单快照](/leetcode/top-100-liked) 选择当天计划内的 `3-5` 题。
2. 每题先写暴力思路和复杂度，再推导优化思路。
3. Python 解法优先追求可复述性，不写过度压缩的一行流。
4. 记录关键不变量、边界条件和失败原因。
5. 当天结束后更新题单状态：未开始、已做、需复盘、已复盘。

## 评估标准

- 能否在不看答案的情况下讲清楚状态定义、转移或不变量。
- 能否写出时间复杂度和空间复杂度，并说明优化来源。
- 是否记录了边界条件，例如空输入、重复元素、越界、负数、环。
- 是否能把同类题归纳为一个可复用模板，而不是只记单题答案。

# 技术学习计划

技术学习计划用于沉淀 AI/ML/Agent 主线知识。每个技术 Day 页面都要按“教材式章节”写作，而不是只列提纲。

## 写作标准

- 先讲问题背景，再给形式定义。
- 所有公式必须先定义符号和 shape。
- 推导不能跳步，尤其是矩阵求导、链式法则和 tensor shape 变化。
- 每个技术主题必须包含最小代码映射，说明公式中的对象对应哪个变量。
- 每页都要包含常见错误、自测题和面试表达。
